# Code of Conduct

Ohiyo follows the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/), version 2.1.

In short: be kind, assume good faith, and help keep this a welcoming space for everyone — warm in the chrome, straight about the stakes. 🐭

Concerns can be reported privately to the maintainers at **conduct@ohiyo.gg**. Reports are handled confidentially.
