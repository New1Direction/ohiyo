Ohiyo typography — mono/tech

More chrome, more signal.

These test monospace and sharper tech fonts. Some are too intense for full chat, but they can be great for buttons, status pills, voice controls, code-ish trust/security moments, and the Ohiyo “premium chrome” identity.

Important:

I would not set the whole message feed in a mono font. Best combos are usually: clean sans for messages + mono for badges/status/buttons, or a sharp display font with Inter for chat.
