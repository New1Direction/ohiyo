Under the hood

A cozy room with a steel door.

Warm inside, sealed shut. We didn't invent the crypto — we used the real, audited thing, and built the server to be blind by design.

The server moves sealed envelopes between devices, enforces membership, and stores nothing it can read — proven by a Rust integration suite that shows a removed member loses access and a pinned identity can't be silently swapped.

HandshakeX3DH

Async key agreement — message someone offline, forward-secret from the first byte.

1:1 sessionsDouble Ratchet

Per-message keys with forward secrecy & post-compromise recovery, via audited libsignal.

GroupsSender keys + epochs

Every membership change rekeys; a removed member reads zero future messages.

TrustSafety numbers

Out-of-band verification across a contact's devices; a swapped key raises a loud banner.
